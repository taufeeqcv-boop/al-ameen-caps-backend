# Fastway reference sources

Official PDFs used to maintain Fastway docs in this repo:

| File | Description |
|------|-------------|
| **CPT RS_2025 ewallet only.pdf** | Cape Town eWallet pricing effective 2 June 2025 (local, short haul, national, satchels, excess labels, zones). |
| **FW System Capabilities - 2023.pdf** | System overview: eCommerce solutions, track & trace, eWallet/customer portal, API integration, dedicated CSD. |

Summaries are kept in:

- `FASTWAY_EWALLET_PRICING_2025.md` – pricing tables and notes
- `FASTWAY_SYSTEM_OVERVIEW.md` – capabilities and API

Keep the PDFs (e.g. in Downloads or a `reference/` folder) for zone maps and full terms. For API access, email it.support@fastway.co.za with API key and business name.
