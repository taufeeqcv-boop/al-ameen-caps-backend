# Static assets for Edge Function branding

Place the **Al-Ameen Emerald Star** logo here for the heritage image branding Edge Function:

- **Filename:** `al-ameen-star-emerald.png`
- **Format:** Transparent PNG (recommended size 200–400px on the longer side).

Once deployed, the site will serve it at:

`https://alameencaps.com/static/al-ameen-star-emerald.png`

Set this URL as **LOGO_URL** in the Supabase Edge Function **process-heritage-image** (Dashboard → Edge Functions → process-heritage-image → Settings).

See **docs/HERITAGE_IMAGE_BRANDING.md** for full setup.
